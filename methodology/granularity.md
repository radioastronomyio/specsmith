# Granularity

The spec can change precision without changing shape.

---

## 1. Purpose

This document explores how the same outcome-to-verification contract works across different levels of specificity. A lightweight spec and a compliance-grade spec use the same format; the depth of the outcome definition and the breadth of the test gate set scale up.
